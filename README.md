# 雾都雨夜 - 文字冒险游戏

一款悬疑推理题材的网页文字冒险游戏。

## 🎮 游戏简介

玩家扮演一名失忆的私家侦探，在一个雨夜接到神秘委托，调查一起富商"自杀"案。通过与不同 NPC 对话、搜集线索、做出选择，逐步揭开真相。

## 🚀 如何运行

直接双击 `index.html` 文件，用浏览器打开即可游玩。

**推荐浏览器**：Chrome / Edge / Firefox 等现代浏览器

## 🎯 游戏特色

- **多分支剧情**：你的每个选择都会影响故事走向
- **5种结局**：根据选择和线索收集情况，解锁不同结局
- **线索系统**：搜集物证、证言、档案，推理真相
- **属性系统**：生命值、理智值，影响游戏进程
- **好感度系统**：与NPC的关系会影响关键信息的获取
- **存档功能**：5个存档位 + 自动存档
- **雨天氛围**：雨滴粒子效果 + 雷声音效

## 🎹 操作方式

| 操作 | 说明 |
|------|------|
| 鼠标点击 | 点击对话框推进剧情、选择选项 |
| 空格 / 回车 | 快速推进对话 |
| ESC | 打开设置 |

## 🎨 关于美术素材

本游戏使用纯CSS绘制的简约风格角色立绘和渐变背景，无需额外图片资源。
如果需要替换为真实的立绘和背景图，可以修改 `game.js` 中的场景数据，
在 `scenes` 对象中添加 `backgroundImage` 和 `characterImage` 字段。

## 💾 存档说明

游戏存档保存在浏览器的 localStorage 中，清除浏览器数据会丢失存档。

## 📝 剧情说明

- 第一章：雨夜委托 - 接受案件，了解基本情况
- 第二章：别墅调查 - 勘察现场，收集线索
- 第三章：深入调查 - 多条调查线，挖掘真相
- 第四章：真相逼近 - 对峙环节，指认凶手
- 终章：5种结局 - 真相大白 / 沉默的正义 / 迷雾重重 / 坠入深渊 / 庸碌人生
